# Methylation Accuracy Analysis and ASCII Decoding Pipeline

Pipeline for methylation accuracy analysis, binary reconstruction, ASCII decoding, and error-rate analysis using methylation-tagged nanopore BAM files.

---

# Overview

This workflow analyzes methylation-tagged BAM files generated from multiple nanopore methylation-calling pipelines, including:

* Dorado HAC methylation calling
* Dorado SUP methylation calling
* DeepMod2 methylation calling
* Guppy HAC methylation calling

The workflow extracts methylation positions from methylation-tagged BAM files and evaluates the accuracy of methylation-based data encoding by reconstructing binary and ASCII representations from methylation patterns.

The workflow performs:

* methylation pileup generation
* CpG methylation extraction
* thresholding and filtering
* methylation position extraction
* binary reconstruction
* ASCII decoding
* error-rate analysis
* bitwise comparison against expected encoded messages

The workflow corresponds specifically to:

| Step | Script                   | Purpose                                        |
| ---- | ------------------------ | ---------------------------------------------- |
| 1    | `run_pipeline_docker.sh` | Main methylation analysis pipeline             |
| 2    | `get_stats.py`           | Computes methylation thresholds                |
| 3    | `meth_analysis.py`       | Converts methylation positions into ASCII      |
| 4    | `meth_parse.py`          | Parses methylation positions into binary/ASCII |
| 5    | `process_methylation.py` | Simplified methylation parser                  |
| 6    | `error_stats.py`         | Computes decoding accuracy statistics          |

---

# Workflow Purpose

This pipeline evaluates whether methylation patterns generated by nanopore methylation callers correctly preserve encoded binary information stored in designed CpG methylation sites.

The workflow assumes:

* specific CpG sites correspond to binary positions
* methylated sites represent binary `1`
* unmethylated sites represent binary `0`

The decoded binary values are grouped into:

* 7-bit ASCII
* or 8-bit ASCII

to reconstruct encoded text.

# Main Analysis Workflow

## Main Pipeline Script

```text id="jlwmvw"
run_pipeline_docker.sh
```

The pipeline performs the following sequence:

1. Detect methylated CpG sites using Modkit
2. Generate methylation pileups
3. Extract CpG methylation statistics
4. Filter methylated positions
5. Compute methylation thresholds
6. Generate methylation-position files
7. Convert methylation patterns into binary
8. Decode ASCII representations
9. Compute error statistics

---

# Main Modkit Command

```bash id="jlwm9m"
modkit pileup \
    --cpg \
    --mod-thresholds C:0.0 \
    --ref reference.fasta \
    sample_mod_sorted.bam \
    output.txt
```

---

# Important Modkit Parameters

| Parameter                | Description                     |
| ------------------------ | ------------------------------- |
| `--cpg`                  | Restricts analysis to CpG sites |
| `--mod-thresholds C:0.0` | Includes all methylation calls  |
| `--ref`                  | Reference genome                |
| `sample_mod_sorted.bam`  | Input methylation BAM           |
